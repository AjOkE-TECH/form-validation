# Contact Form with Validation (HTML, CSS, JavaScript, PHP)
## Project Description
This project is a simple Contact Form built using HTML, CSS, JavaScript, and PHP.
It demonstrates:
- Client-side validation using JavaScript
- Server-side processing using PHP
- Form submission using POST method
- Error handling 
The goal of this project is to understand how frontend validation works before sending data to the server.
## Technologies Used
- HTML5 – Structure of the form
- CSS – Styling and layout (Grid)
- JavaScript – Client-side form validation
- PHP – Server-side form processing
